# 💰 DevFinance - Controle de Gastos

Dashboard financeiro para gestão de entradas e saídas com cálculo automático de saldo e persistência de dados.

### 🛠 Tecnologias e Conceitos:
- **Lógica de Programação:** Uso de `map`, `filter` e `reduce` para processar valores.
- **Persistência de Dados:** Uso de `LocalStorage` para manter os dados após atualizar a página.
- **CSS Moderno:** Layout responsivo com CSS Grid e Flexbox.
